# Guide de Suivi SEO Mensuel - Studioriad

Ce guide explique comment extraire les données de Google Search Console (GSC) pour que votre assistant IA puisse les analyser et optimiser votre référencement.

## 1. Configuration des Alertes (À faire une fois)

Pour être prévenu immédiatement en cas de problème technique (ex: page 404, problème mobile) :

1.  Connectez-vous à [Google Search Console](https://search.google.com/search-console).
2.  Cliquez sur votre **Avatar** (haut droite) > **Paramètres d'utilisateur**.
3.  Cliquez sur **Préférences e-mail**.
4.  Activez **"Toutes les notifications"**.

## 2. Le Rituel Mensuel (Extraction des données)

Chaque début de mois, suivez cette procédure pour obtenir les données brutes :

### Étape A : Exporter les Performances
1.  Allez dans l'onglet **Performances** (menu de gauche).
2.  Assurez-vous que la période est réglée sur **"3 derniers mois"** (pour voir la tendance).
3.  Cliquez sur **Exporter** (bouton en haut à droite).
4.  Choisissez **Google Sheets** (plus lisible) ou **CSV**.

### Étape B : Analyser avec l'IA
Revenez vers votre assistant (moi) avec les informations suivantes copiées depuis l'export :

1.  **Top Requêtes (Queries)** : Copiez les 20 premiers mots-clés qui vous apportent du trafic (ou des impressions).
2.  **Top Pages** : Copiez la liste des pages les plus visitées.
3.  **Tendance** : Dites-moi si la courbe globale (clics/impressions) monte ou descend.

**Exemple de prompt pour l'IA :**
> "Voici mes stats GSC du mois. 
> Top mots-clés : [coller liste]
> Top pages : [coller liste]
> Analyse ça et dis-moi quel article de blog je devrais écrire pour booster le trafic."

## 3. Points de vigilance (KPIs)

Voici les indicateurs que nous surveillerons ensemble :

*   **Impressions** : Combien de fois Studioriad apparaît dans Google (Visibilité).
*   **CTR (Taux de clic)** : Si ce chiffre est bas (< 2%), nous devrons réécrire vos titres (meta titles).
*   **Position Moyenne** : Si vous êtes bloqué en page 2 (position 11-20), nous ferons une optimisation ciblée.
